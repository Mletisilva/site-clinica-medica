🏥 Site Institucional - Clínica Médica - Projeto desenvolvido para fins educacionais.

Projeto de desenvolvimento de um site institucional para uma clínica médica, criado como parte do curso de HTML da DIO.
O objetivo do projeto foi aplicar conceitos fundamentais de estruturação de páginas web, organização semântica e navegação entre páginas utilizando HTML e CSS.


📄 Páginas do Projeto
O site é composto por:

- Home (home.html) – Página inicial com apresentação da clínica.
- Sobre Nós (sobre-nos.html) – Informações institucionais.
- Horário de Atendimento (horario-atendimentos.html) – Dias e horários disponíveis.
- Contato (contato.html) – Informações de contato e formulário.
- Estilização (base.css) – Arquivo responsável pelo design e layout.
- Imagens – Pasta contendo os recursos visuais do site.


🛠️ Tecnologias Utilizadas:

- HTML5
- CSS3


🎯 Objetivo do Projeto

- Praticar estrutura básica de páginas web.
- Aplicar organização semântica em HTML.
- Criar navegação entre múltiplas páginas.
- Desenvolver um layout simples e funcional.
